# Trading Bot - Binance Futures Testnet
### Web UI + CLI Interface

A production-ready trading bot with both **Streamlit Web UI** and **Command Line Interface** for Binance Futures Testnet.

## 🌟 Features

- ✅ **Streamlit Web Interface** - Beautiful, responsive UI
- ✅ **CLI Interface** - For automation and scripts  
- ✅ Market & Limit Orders
- ✅ Real-time price feeds
- ✅ Order history tracking
- ✅ Professional logging system
- ✅ Comprehensive error handling
- ✅ Account balance viewer
